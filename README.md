# Sentiment Analyzer

A modern, beautiful frontend application for AI-powered text analysis that supports multiple input types including text, audio files, PDFs, and web URLs.

## Features

### 🎯 Multiple Analysis Types
- **Text Analysis**: Direct text input with comprehensive AI insights
- **Audio Analysis**: Upload audio files for transcription and analysis
- **PDF Analysis**: Extract and analyze content from PDF documents
- **URL Analysis**: Analyze web page content from any URL

### 🎨 Modern UI/UX
- Beautiful gradient design with glass morphism effects
- Dark/light theme toggle
- Responsive design for all devices
- Smooth animations and transitions
- Loading states and progress indicators
- Drag-and-drop file uploads

### 🚀 Technology Stack
- **Frontend**: Next.js 15 with App Router
- **Styling**: Tailwind CSS 4 with shadcn/ui components
- **Icons**: Lucide React
- **Theme**: next-themes for dark/light mode
- **TypeScript**: Full type safety
- **Backend Integration**: FastAPI compatible endpoints
